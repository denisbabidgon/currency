


git init . - инициализировать новый гит репозиторий (локальный)
git status - посмотреть текущий статус проекта (файлов)
git add . или вместо точки перечисляем файлы которые мы хотим добавить в "текущий коммит"

git commit -m 'текст комментария' - закоммитить изменения

git push - отправить изменения в удаленный репозиторий (если с этим репозиторием есть связь)

git branch имя_ветки - создать новую ветку
git checkout имя_ветки - переключиться на указанную ветку
